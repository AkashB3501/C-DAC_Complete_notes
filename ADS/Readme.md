### Steps to understand the ADS programs:

  1. **Break Down the Problem** – identify input, operations, and expected output.
  2. **Understand the Data Representation** – array, linked list, stack, queue, tree, graph, etc.
  3. **Trace the Functions One by One** – check parameters, return values, and variable updates.
  4. **Dry Run with Small Input** – follow step by step and update variable values.
  5. **Notice Control Flow** – observe loops, conditions, and recursion.
  6. **Connect Code with Real-World Analogy** – relate DS behavior to everyday examples.
  7. **Keep Parallel Notes** – code snippet, dry-run table, and structure diagram.
